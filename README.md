# AR Manual App

A mobile AR application built with Unity that allows users to scan QR codes to load interactive AR manuals with marker-based tracking. 

A mobile application providing functionality for viewing manuals and editing existing interaction points.

Part of a bachelor's thesis.

## Tech Stack

**Unity**

[**ArUcoDetector**](https://github.com/CoolLikeGoose/ArUcoDetector) - Used for detection and PnP solving as native plugin.

[**ZXing**](https://github.com/zxing/zxing) - Used for QR reading.
